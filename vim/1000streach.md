



vim体操


## まずは上下の運動から〜

```sh
jjkkjjkk
```

## 次に左右の運動〜

```sh
hhhhlllllhhhhhllll
```

## さらに大きく上下の運動

```sh
HMLHMLHMLHML
```

左右の動きをしていたHとLが上下に大きな動きになるのが特徴的ですね〜

## スクロールの運動

```sh
Ctrl+e,Ctrl+y
```

## 組み合わせスクロールの運動

画面を大きく移動するHとLを組み合わせて動かしましょう。

```sh
HkkkkkkkLjjjjjjjHkkkkkkkkLkkkkkkkk
```




## Linuxコマンドの実行

`;!l`に続いてコマンドを入力することでlinuxコマンドの実行が可能。

```sh
;!lls
```

```sh
;!ldate
```
