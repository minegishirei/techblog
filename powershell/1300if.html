powershell 入門 学習サイト 〜変数編〜 h2> 条件文(if文) if文の書き方は以下の通り 

$condition = $true 
if ($condition ){
    Write-Output "The condition was true" 
} else {
    Write-Output "The condition was false"
    $condition = $false
}
$condition = $false
if ($condition ){
    Write-Output "The condition was true"
}else{
    Write-Output "The condition was false"
}

if文の同値評価は「-eq」 


$num1 %3D2 if ($num1 -eq 2 ){

    Write-Output "The condition was false"
}
