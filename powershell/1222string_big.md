## 大文字小文字変換

.ToUpper(),.ToLower()メソッドを使おう

### 大文字化

サンプル
```ps1
"Hello World".ToUpper()
```

実行結果

```ps1
HELLO WORLD
```

### 小文字化

サンプル

```ps1
"Hello World".ToLower()
```

```ps1
hello world
```
