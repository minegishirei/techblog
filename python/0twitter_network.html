Q. twitter network analyticsとはなんでしょうか?

>A.過去の会話記録をもとにアカウント同士の関係図を可視化するツールです

本サイトはtwitter上でのアカウントとアカウントの人脈を可視化します。

また、可視化された人脈図を俯瞰してみることで、これまで気づかなかった身近な発見や新しいつながりのきっかけになることを目指しています。

Q.人脈図をどのように見れば良いでしょうか

A.特定の見方はありませんが人脈図をみるときは以下の点について注意してみてください。

1、人脈図には塊ができている箇所がありますか？その塊の中心にはどのアカウントがいるのでしょうか？

[ 2、ネットワークが小さい方は三角形ができていないか注目してみてください。

Q.作成ボタンを押したのに何も出てきません。なぜですか?

>A.作成ボタン押下後のページには何も表示されない場合、以下の理由が考えられます。

1、直近のその他のアカウントとの会話の回数が少ない(本サイトは会話の記録をもとにネットワークを構成します)

2、間違えたアカウント名を入力している

3、鍵がついたアカウントで外部からの参照が不可能

Q.動作が遅いです。スムーズに見れないでしょうか?

A.スマートフォンや古いパソコンなど端末によっては動作が遅くなる可能性があります。

本ネットワークの動作は端末のCPUの精度に依存します。

スムーズにみたい方はパソコンでの視聴をおすすめします。
