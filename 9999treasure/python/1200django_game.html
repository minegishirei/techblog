1. Fork and clone this repository 

2. Create a Python virtual environemnt 

3. In that environment, run pip install -r requirements. txt inside your project 

4. Install node modules with npm install 

5、Create the local database with python manage.py migrate

6. Run webpack to build the components with npm run webpack

7、Run Django development server on port 8080 - python manage.py runserver 8080
