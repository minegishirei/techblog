


## go言語の変数宣言

### 明示的な型宣言

通常の変数宣言では `var 変数名 型名`で宣言する。

```go
package main

import "fmt"

// this is main function only print "Hello World"
func main() {
	// int型の変数nを定義・代入
	var n int
	n = 5
	fmt.Println(n)

	// int型の変数x,y,zを定義・代入
	var x, y, z int
	x, y, z = 1, 3, 5
	fmt.Println(x, y, z)
}
```
